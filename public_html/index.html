Security Library

As my collection of bookmarks, tools and referencework for IT & cyber security grew bigger, I also grew a bigger need to store them all in one convenient place. For that reason I created the Security Library. My plan is for this to be a collection of links to useful tools for hacking/CTF challenges, documentation and references. Additionally just small bits of info that I wish I didn't constantly have google again. You could see it as my personal notebook, that everyone is free to use if they want.
