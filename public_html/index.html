My plan is to create a large collection of documentation and more that I would not be able bookmark effectively but still want to store somewhere. You could see it as my personal technical documentation/wiki, that's hosted online. It also comes with a Youtube channel where I will post videos, and this will temporarily link there until the library is finished.
