Security Library

As my collection of bookmarks, PDF's and tools grew bigger, I also grew a need to store them all in one place that I could reach from multiple locations. For that reason I created my project the Security Library. My plan is for the site to be a big collection of info I end up finding interesting enough to permanently bookmark or end up needing often. You could see it a bit as my personal technical documentation, that everyone is free to use if they find a use for it.
